 GreenLeaf Website  
A responsive, accessible, and interactive website created for the POE Final Assessment.  
GreenLeaf is an eco-friendly landscaping and gardening brand.  
This project demonstrates clean web development practices using HTML, CSS, and JavaScript, including SEO, forms, dynamic features, maps, and full documentation.



Website Description  
The GreenLeaf website aims to present a modern, user-friendly layout that showcases landscaping services, eco-friendly gardening tips, and easy access to enquiries and contact information.

The website includes:  
- Home Page  
- About Page  
- Services Page  
- Gallery (with Lightbox)  
- Enquiry Page (with JS Validation)  
- Contact Page (with Google Map + Email Sending)
